# siteportal-leyu-resource

## 项目简介

`siteportal-leyu-resource` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单一域名或具体网站，仅作为静态 HTML 资源的集中存放位置，便于版本管理与分发。

## 目录结构

```
siteportal-leyu-resource/
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── docs/           # 文档或说明文件
└── README.md       # 本文件
```

目录结构可能随仓库更新而调整，请以实际文件为准。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面之间无强依赖关系，可单独使用或部署。
- 归档内容仅包含页面本身及其所需的资源文件，不包含任何后端服务或动态数据。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的 HTML 文件即可查看页面内容。
3. 如需部署，可将 `pages/` 及 `assets/` 目录上传至任意静态文件服务器。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面内容。
- 如发现页面无法正常显示或资源缺失，可提交 Issue 反馈。
- 欢迎提交 Pull Request 贡献新的页面或改进现有内容，但请确保所提交的页面为独立、完整的 HTML 文件。

本仓库内容仅供学习与参考使用，具体许可信息请参见仓库内的 LICENSE 文件（如有）。
